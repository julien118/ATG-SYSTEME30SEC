# Assistant de Visite IONNYX — Version Démo

## Contexte
Ce projet est une version démonstration de l'Assistant de Visite Terrain IONNYX.
Le PRD complet est dans `PRD.md` — lis-le intégralement avant toute action.

- `npm run build` — build production
